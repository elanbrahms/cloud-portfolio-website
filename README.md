\# Cloud Portfolio Website



A static portfolio website hosted on AWS using Amazon S3 and CloudFront.



\## Services Used

\- Amazon S3

\- Amazon CloudFront

\- AWS IAM



\## Features

\- Static website hosting

\- Cloud delivery through CloudFront

\- Project showcase page



\## Project Purpose

This project was built to strengthen hands-on AWS skills related to static hosting, content delivery, and cloud architecture.
